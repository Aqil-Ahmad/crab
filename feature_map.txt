# Grid Image Editor - Feature Map
# This file documents all features and their navigation paths for AI integration
# Format: FEATURE_NAME | NAVIGATION_PATH | KEYBOARD_SHORTCUT (if any) | PARAMETERS (if any)

================================================================================
FILE OPERATIONS
================================================================================

New Image | File > New | Ctrl+N | width, height, background (Black/White/Transparent/Custom)
Open Image | File > Open | Ctrl+O | file_path (supports: .png, .jpg, .jpeg, .bmp)
Export Image | File > Export.. | Ctrl+Shift+S | file_path (supports: .png, .jpg, .jpeg, .bmp)

================================================================================
EDIT OPERATIONS
================================================================================

Undo | Edit > Undo | Ctrl+Z | none
Redo | Edit > Redo | Ctrl+Y | none

================================================================================
VIEW OPTIONS
================================================================================

Toggle Menu Bar | View > Menu bar | Alt | toggle
Transparent Checker Shader | View > Use Transparent Checker Shader | none | toggle

================================================================================
CANVAS NAVIGATION
================================================================================

Reset Canvas Navigation | none | Ctrl+Shift+R | none
Zoom In/Out | Canvas (hover) + Mouse Scroll | Mouse Scroll | zoom_factor
Pan Canvas | Canvas (hover) + Middle Mouse Drag | Middle Mouse | pan_delta_x, pan_delta_y

================================================================================
TOOLS
================================================================================

No Tool | Tools Panel (click icon) | N | none
Move Tool | Tools Panel (click icon) | V | none
Brush Tool | Tools Panel (click icon) | B | size (1-2000px), hardness (0-100%), opacity (0-100%)
Eraser Tool | Tools Panel (click icon) | E | size (1-2000px), hardness (0-100%), opacity (0-100%)
Fill Tool | Tools Panel (click icon) | G | tolerance (0-255), opacity (0-100%), contiguous (bool)

Tool Size Decrease | none | [ | decreases current tool size
Tool Size Increase | none | ] | increases current tool size
